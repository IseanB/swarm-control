# swarm-control

## Development Enviorment Setup
1. Clone the repository.
2. Go to the repository.
   - ```cd /path/to/your/project```.
3. Setup a virtual enviorment
  -  ```python3 -m venv env```
  - ```source env/bin/activate```
4. Install dependenceies
  -  ```pip install matplotlib```
  -  ```pip install seaborn```
  -  ```pip install scipy```
5. Run python script
  - ```python3 main.py```
